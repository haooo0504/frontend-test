## 購物車

功能說明：
1. 取得加入購物車的商品資料，並渲染至畫面。
2. 計算總項目數、總金額。
3. 點擊刪除按鈕，刪除購物車內商品，並更新導覽列購物車數量。
4. 點擊成立訂單時，印出購物車內所有已選取商品。資料格式如下：

	```
	params: [
		{
			"id": 1, // 商品id
			"price": 500, // 單一商品價格
			"qty": 1 // 購買數量
		},
		{
			"id": 2,
			"price": 750,
			"qty": 2
		}
	]
	```
